# 21:9 사진 크롭 애플리케이션

사진을 21:9 비율로 자동 크롭하여 다운로드하는 PWA 웹 애플리케이션입니다.

> note: 그랑 콜레오스 바탕화면에 사진 넣을 때 자체 사진앱에서 크롭하는 과정이 번거로워서 이 앱을 만들게 되었습니다.

## 기능

- 📸 사진 업로드 (버튼 클릭 또는 드래그 앤 드롭)
- ✂️ 21:9 비율 고정 크롭
- 🔄 90도 단위 사진 회전
- 🖱️ PC: 마우스 드래그 및 휠 줌
- 📱 모바일: 터치 드래그 및 핀치 줌
- 💾 크롭된 이미지 즉시 다운로드
- 📲 PWA 지원 (모바일 설치 가능)

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm start

# 프로덕션 빌드
npm run build
```

## 사용 방법

1. 사진을 업로드합니다 (클릭 또는 드래그 앤 드롭)
2. 크롭할 영역을 조정합니다
   - 드래그하여 이미지 위치 조정
   - 마우스 휠 또는 핀치로 확대/축소
   - 회전 버튼으로 90도 단위 회전
3. "다운로드" 버튼을 눌러 크롭된 이미지를 저장합니다

## 기술 스택

- React 18
- HTML5 Canvas API
- PWA (Progressive Web App)
- 클라이언트 사이드 이미지 처리
